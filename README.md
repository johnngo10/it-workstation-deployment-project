# 🖥️ IT Workstation Deployment Project

## 📌 Overview
This project simulates a real-world IT support task involving the setup and deployment of a workstation for a client. The objective was to select compatible hardware, install and configure the operating system, and ensure all required software was properly installed and functioning.

---

## 🛠️ Tools & Technologies
- Windows 10 Pro  
- PC Hardware Components (CPU, RAM, Storage, GPU)  
- BIOS/UEFI  
- System Diagnostics & Logs  

---

## ⚙️ What I Did
- Selected and validated compatible hardware components based on business needs  
- Created a detailed pricing quote including hardware and service costs  
- Installed and configured Windows 10 Pro  
- Performed system updates and verified OS integrity  
- Installed and tested required software for end users  
- Verified system functionality using a structured deployment checklist  
- Documented the entire process and communicated updates to the client  

---

## 🧪 Key Skills Demonstrated
- Hardware troubleshooting & setup  
- OS installation & configuration  
- Software deployment  
- System validation & testing  
- Technical documentation  
- Client communication  

---

## 📂 Project Files
- Quote document (hardware + pricing)  
- Workstation deployment checklist  
- Client communication email  
- Supporting screenshots (OS version, updates, installed software)  

---

## ✅ Outcome
Successfully deployed a fully functional workstation that meets client requirements, with all hardware and software tested and verified.
